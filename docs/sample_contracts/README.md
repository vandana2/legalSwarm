# Sample Contracts for Testing

This directory contains sample contracts for testing LegalSwarm's analysis capabilities.

## Usage

1. **NDA.pdf** - Non-Disclosure Agreement (sample)
2. **SaaS_Agreement.pdf** - Software as a Service Agreement (sample)
3. **Vendor_Agreement.pdf** - Vendor Agreement (sample)

## Adding More Samples

Place your PDF contracts in this directory for testing.

## Expected Output

LegalSwarm will analyze these contracts and provide:

- Risk Score (0-10)
- Flagged Clauses with explanations
- Compliance Issues
- Suggested Redlines
- Final Verdict (SIGN / SIGN_WITH_MODIFICATIONS / DONT_SIGN)
